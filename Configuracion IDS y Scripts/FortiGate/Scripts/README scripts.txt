El script procesar_pcapngs.sh contiene los comandos usados para procesar los PCAPNGs con FortiGate.

El script extraelogs.py extrae de los logs de FortiGate del directorio en distintos archivos CSV: uno con campos relevantes (csv), uno específico para alertas L1 (ips), uno específico para alertas L2 (app-ctrl) y uno con todos los campos detectados (csv_full)

El script cuentaattacks.py cuenta las ocurrencias de cada ataque para los archivos CSV L1 (ips) 

El script cuentaapps.py cuenta las ocurrencias de cada ataque para los archivos CSV L2 (app-ctrl)
